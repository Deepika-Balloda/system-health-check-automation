# System Health Check Automation

A Python script that automates system health checks by monitoring CPU, memory, and disk usage, then logs results with timestamps.

## Features
- Monitors CPU, memory, and disk usage
- Generates timestamped reports
- Saves results to `health_report.txt`
- Simple and lightweight automation

## How to Run
1. Install dependencies: pip install psutil
2. Run the script: python health_check.py
   
## Example Output
System Health Check - 2026-04-18 04:45:00
CPU Usage: 12%
Memory Usage: 45%
Disk Usage: 67%
